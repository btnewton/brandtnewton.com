Rep Check

A Powerful One Rep Max Calculator

This project was only meant to be a trial run for publishing an app to Google Play, but it turned into a nice little app. The interface supports an efficient workflow and encourages users to only track their primary exercises to keep sessions short but frequent.

Features

  • Calculate your 1-20 rep max performance.
  • Save sets to a local SQLite Database.
