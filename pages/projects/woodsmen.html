Woodsmen

2D Java Game

Less than half a year after completing my first programming course (taught in MATLAB) I submitted this game, which is still a work in progress, to the 2013 UVM CS Fair. I followed a tutorial series to get the initial framework down and then optimized it and added features. This is a video of the submission.

Since the 2013 UVM CS Fair I have made some changes but it has never been my primary focus. I enjoy creating the graphics and coming up with new ways to increase the average FPS.
